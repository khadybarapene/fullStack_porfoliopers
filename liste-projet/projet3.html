Dashboard Analytics

Description du projet

Dashboard Analytics est une application de visualisation de données développée en utilisant le framework React. Elle permet aux utilisateurs de visualiser et d'analyser des données complexes à travers des graphiques et des tableaux de bord interactifs. L'application offre une interface conviviale et personnalisable pour les transactions, ainsi que des fonctionnalités de gestion des stocks et de suivi des commandes pour les administrateurs.

Technologies utilisées

  • React
  • JavaScript
  • Redux
  • Node.js
  • Express
